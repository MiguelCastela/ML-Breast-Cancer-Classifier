\section*{Introdução}

Este trabalho tem como objetivo desenvolver um modelo de Machine Learning capaz de, a partir de dados obtidos de amostras sanguíneas, identificar de forma precisa os utentes com diagnóstico de cancro da mama.

O dataset considerado neste trabalho é constituído por dados de 116 pacientes, sendo 64 diagnosticados com cancro da mama e 52 pertencentes ao grupo de controlo saudável. As nove variáveis analisadas incluem parâmetros demográficos e bioquímicos, nomeadamente a idade, o índice de massa corporal (BMI), a concentração de glicose e insulina, o valor de HOMA, e os níveis de leptina, adiponectina, resistina e MCP1.

\section*{Métodos e Metodologias}

-partição dos dados

os dados foram divididos em dados de treino (70\%)e dados de teste (30\%)

---

É necessário fazer uma seleção e redução das features de forma a perceber o poder discriminativo das features, i.e., a sua relevância e também a sua redundância.

-kruskal-williams

teste estatístico não paramétrico usado para comparar três ou mais variáveis independentes, que avalia a relevância das features.

Para aplicar este teste usámos a função do python stats.kruskal, que às features vai selecionar as que são classificadas como saudável ou cancerígena.

-ROC, AUC

Método interessante para problemas de duas classes. A curva ROC representa a proproção de deteções positivas corretas (sensibilidade) em função da taxa de falsos positivos (1-especificidade).

-matriz de correlação

Indica como as variáveis variam e co-variam.

-PCA

Tem como objetivo encontrar um conjunto reduzido de novas variáveis com menos redundância, com perda mínima de informação, ou seja, perda mínima de variância.

Projetam-se os dados em direções em que variam mais.

Vetores próprios definem a orientação dos eixos da elipse.
Valores próprios são as variâncias ao longo das direções apontadas.

--> Critério de Scree
quando o plot estabiliza

--> Critério de Kaiser
para dados normalizados, quando um valor próprio vai abaixo de 1


-LDA

O objetivo é é separar as amostras em grupos distintos colocando-os num espaço maximizando a separabilidade entre classes e minimizando a variabilidade dentro da mesma classe. A ideia é projetar os dados de acordo com estas condições.

-distância euclidiana

\[
||x-m||_e = \sqrt{\sum_{i=1}^{d} (x_i - m_i)^2}
\]

-mahalanobis

Métrica sofisticada que se ajusta a vários formatos de distribuição de dados.

\[
||x-y||_m = \sqrt{(\mathbf{x} - \mathbf{y})^{T} \, \mathbf{C}^{-1} \, (\mathbf{x} - \mathbf{y})}
\]

-fisher LDA

Este classificador combina o método LDA com o método de Mahalanobis.
